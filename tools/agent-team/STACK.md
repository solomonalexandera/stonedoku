# Stack summary
- Frontend: Vanilla HTML/CSS/JS (app.js, styles.css, index.html), Firebase client SDK.
- Backend: Firebase (Hosting, Firestore, Realtime Database, Storage), Cloud Functions (TypeScript in functions/).
- Testing: Playwright E2E specs under tests/playwright.

# Repo map
- Frontend/app shell: index.html, app.js, styles.css, assets/, effects.js.
- Firebase config/rules: firebase.json, firebase/firestore.rules, firebase/database.rules.json, firebase/storage.rules.
- Functions: functions/src/*.ts (api.ts, vanity.ts, mailer.ts, firebaseAdmin.ts).
- Tests: tests/playwright/*.spec.js.
- Design brief: DESIGN_BRIEF.md
